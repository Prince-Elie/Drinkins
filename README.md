# Drinkins

🍹 Une application web moderne avec animations GSAP spectaculaires.

## 🚀 [Voir la démo live](https://prince-elie.github.io/Drinkins/)

---

## À propos

Drinkins est un projet web interactif mettant en avant des animations fluides et une interface moderne.

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)

## 📋 Installation

1. Clonez ce repository :
```bash
git clone https://github.com/prince-elie/Drinkins.git
```

2. Ouvrez `index.html` dans votre navigateur

## 📱 Responsive

L'application est entièrement responsive avec une navigation mobile optimisée.
